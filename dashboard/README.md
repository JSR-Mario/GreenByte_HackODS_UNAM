# dashboard/

Esta carpeta contendrá el tablero final del proyecto GreenByte.

## Formato previsto

- `index.qmd` — Tablero principal en Quarto Dashboard
- O bien: `greenbyte_dashboard.ipynb` — Libreta Jupyter con visualizaciones y narrativa

## Estructura narrativa del tablero (borrador)

1. **Introducción / Contexto** — ¿Qué es el colapso ecosistémico compuesto? ¿Por qué México?
2. **Tendencias nacionales** — Evolución 2015–2024 de variables clave (Mann-Kendall)
3. **Señal climática vs. ENSO** — Descomposición de SST: qué es calentamiento real vs. ruido ENSO
4. **Mapa de hotspots** — Distribución geográfica del índice de multi-estrés (Gi*)
5. **Comparativa regional** — CDMX · Monterrey · Cuenca Pérmica (Texas)
6. **Conclusiones y acción** — Qué dicen los datos para el ODS 13

## En construcción

El tablero se irá construyendo a medida que avance el análisis EDA.
